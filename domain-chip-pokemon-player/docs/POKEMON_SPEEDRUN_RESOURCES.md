# Pokemon Red Speedrun Resources

Use these in this order when teaching the chip.

## Tier 1: Ground Truth

- `https://www.speedrun.com/pkmnredblue`
  - leaderboard, category rules, current top runs, and community truth surface
- `https://www.speedrun.com/pkmnredblue/guides`
  - route, menuing, and beginner-to-advanced guide hub

## Tier 2: Mechanics And Task Extraction

- `https://github.com/pret/pokered`
  - disassembly-level source for maps, scripts, text flow, encounters, and event logic
- `https://pokemon-speedrunning.github.io/Gen1ClassicDVCalculator/index.html`
  - practical manipulation/tooling surface for Gen 1 route-critical stat reasoning

## Tier 3: Emulator And Automation

- `https://github.com/Baekalfen/PyBoy`
  - emulator automation surface used by this chip

## How To Use Them

- Use `speedrun.com` to decide what strong runs look like now.
- Use the guides hub to extract route fragments, benchmark tasks, and menu doctrine.
- Use `pret/pokered` when the chip needs map/script/mechanic truth instead of commentary.
- Use the DV calculator when route-critical Gen 1 manipulation logic matters.
- Use PyBoy docs/source when turning route knowledge into executable benchmark tasks.
